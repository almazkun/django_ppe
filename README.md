# django_ppe
Django PPE

## Installation
Pipenv is used to manage the virtual environment and dependencies. To install pipenv, run the following command:
```bash
pip install pipenv
pipenv install
```

## Running the server
To run the server, run the following command:
```bash
pipenv run python manage.py runserver
```
